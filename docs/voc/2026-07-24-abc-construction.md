# VOC: ABC건설 — 다크 모드 UI 일부 영역 라이트 테마 색상 표시 문제

- **고객사**: ABC건설
- **접수 경로**: 고객 문의 이메일 (VOC)
- **관련 Task**: https://app.notion.com/p/e038a24187f848df811b7d5fd2d56181

## 배경 / 고객 원문 핵심

"다크 모드 사용 시 특정 영역이 라이트 테마 색상으로 남아 가독성이 크게 떨어져 메일 드립니다."

Web(Chrome) 다크 모드 사용 시 특정 UI 영역이 라이트 테마 색상으로 남아 가독성이 저하된다는 VOC입니다. 재현 경로 및 구체적 영향 영역은 추가 확인이 필요합니다.

## 현재 문제

- 다크 모드 활성화(Web/Chrome) 환경에서 일부 UI 컴포넌트가 라이트 테마 색상(밝은 배경·텍스트)으로 렌더링됨
- 해당 영역에서 텍스트 가독성이 크게 저하되어 사용자 경험에 부정적 영향
- 다크 모드 테마 토큰 또는 CSS 스타일링이 누락되었을 가능성 있음

## 제안 변경사항

1. **재현 확인**: 고객 환경(Web/Chrome 다크 모드)에서 동일 증상 재현 여부 확인 및 영향을 받는 UI 영역 식별
2. **원인 분석**: 다크 모드 테마 토큰 / CSS custom properties 적용 누락 여부 점검
3. **수정 방향 결정**: 누락된 토큰 추가 또는 컴포넌트별 다크 모드 스타일 보완
4. **추가 정보 요청(필요 시)**: 고객에게 문제 발생 영역 스크린샷 및 페이지 경로 요청

## 우선순위 / 참고

- **체감 우선순위**: 중간 (가독성 저하로 사용 불편이 명확하나 기능 자체는 동작함)
- **판단 근거**: 다크 모드 UI 렌더링 이슈로 사용자 경험에 직접 영향; 재현 범위에 따라 우선순위 조정 필요
- **추가 확인 필요**: 영향을 받는 구체적 페이지/컴포넌트 목록, Chrome 버전, OS 다크 모드 vs 앱 내 설정 여부
