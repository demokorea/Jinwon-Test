# VOC: ABC건설 — 다크 모드 사용 시 일부 UI 영역 테마 색상 오류

- **고객사**: ABC건설
- **접수 경로**: 고객 문의 이메일 (VOC)
- **관련 Task**: https://app.notion.com/p/72b90accb1154614a2b553f8f69144ca

## 배경 / 고객 원문 핵심

"다크 모드 사용 시 특정 영역이 라이트 테마 색상으로 남아 가독성이 크게 떨어져 …"

Web(Chrome) 환경에서 다크 모드를 사용할 때 좌측 사이드바 등 특정 메뉴/뷰 영역이 라이트 테마 색상으로 렌더링되어 가독성이 현저히 저하된다는 VOC입니다.

## 현재 문제

- Web(Chrome) 다크 모드 환경에서 일부 UI 영역의 배경색·텍스트색이 라이트 테마 값으로 고정됨
- 해당 영역의 재현 경로(좌측 사이드바 특정 메뉴/뷰)가 아직 구체적으로 확정되지 않은 상태

## 제안 변경사항

1. 고객이 언급한 재현 경로(좌측 사이드바 특정 메뉴/뷰)에 대한 추가 정보 요청 여부 검토
2. Web(Chrome) 다크 모드에서 재현 테스트 실시 및 영향 범위 파악
3. 다크 모드 테마 CSS 변수 혹은 컴포넌트별 테마 적용 누락 부분 확인 후 수정
4. 수정 계획 수립 및 배포 일정 산정

## 우선순위 / 참고

- **체감 우선순위**: 중 (가독성 영향으로 사용성에 직접 지장을 주지만, 기능 자체는 동작함)
- **판단 근거**: 다크 모드는 선택적 설정이나, 이를 활성화한 사용자에게는 매 사용 시 불편함이 발생
- **추가 확인 필요**: 재현 경로를 고객에게 재확인 필요 (좌측 사이드바의 정확한 메뉴명·뷰 이름)
