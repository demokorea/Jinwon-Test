# VOC: PAL건설 — 가이드 문서 링크 404

- **고객사**: PAL건설
- **접수 경로**: 고객 문의 이메일 (VOC)
- **관련 Task**: https://app.notion.com/p/ac5cd5ad540e4af897f6850d8c63f625

## 배경 / 고객 원문 핵심

"노션을 사내 위키로 사용 중인데, 가이드 문서 내 링크가 잘못 연결되어 404로 이동하는 문제가 있어 메일 드립니다."

Web(Chrome)과 Desktop App에서 동일하게 재현됨.

## 현재 문제

사내 위키로 사용 중인 Notion 가이드 문서 내 특정 링크가 404 페이지로 이동함.  
Web 브라우저(Chrome)와 데스크톱 앱 양쪽에서 동일하게 재현되어, 단순 캐시/브라우저 문제가 아닌 실제 링크 오류로 판단됨.

## 제안 변경사항

1. **링크 대상 특정**: 어떤 가이드 문서의 어느 링크가 404를 반환하는지 정확한 URL/경로 확인
2. **리다이렉트 및 목적지 점검**: 재현 경로 기준으로 링크 목적지(리다이렉트 포함) 전수 점검 및 수정
3. **문서 업데이트**: 잘못된 링크를 올바른 URL로 교체하거나 삭제
4. **내부 공지/릴리즈 노트**: 수정 후 고객에게 안내할 수 있도록 내부 공지 또는 릴리즈 노트 작성 필요 여부 확인

## 우선순위 / 참고

- **체감 우선순위**: 중간 (High) — 사내 업무 도구로 활용 중인 가이드 문서 접근 불가 → 업무 저해 가능성
- **판단 근거**: Web/Desktop 양쪽에서 재현되므로 클라이언트 환경 문제가 아닌 문서 또는 서버 측 링크 오류
- **추가 확인 필요**:
  - 고객으로부터 정확한 404 URL 또는 재현 경로 수집 필요
  - 해당 링크가 외부 URL인지, Notion 내부 페이지 링크인지 구분 필요
