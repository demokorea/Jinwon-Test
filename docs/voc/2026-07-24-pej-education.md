# VOC: PEJ교육 — DB 설정 '자동 알림' 도움말 문구가 이메일 발송으로 오해 유발

- **고객사**: PEJ교육
- **접수 경로**: 고객 문의 이메일 (VOC)
- **관련 Task**: https://app.notion.com/p/64edab9f50a34fe39ce27fc9b98fd6c7

## 배경 / 고객 원문 핵심

"자동 알림을 켜면 모든 변경 사항이 이메일로 발송됩니다."라는 도움말 문구 때문에 혼동이 있습니다.  
실제 동작은 이메일이 아니라 **인앱(In-App) 알림** 중심으로 동작합니다.

## 현재 문제

DB 설정 > 알림 > '자동 알림' 토글의 도움말 문구가 실제 동작과 불일치합니다.

- **현재 문구**: "자동 알림을 켜면 모든 변경 사항이 이메일로 발송됩니다."
- **실제 동작**: 이메일이 아닌 인앱 알림(In-App Notification)으로 전달됨
- **영향**: 사용자가 이메일을 기대하고 기다리거나, 알림이 오지 않는다고 오해하는 혼동 발생

## 제안 변경사항

1. **한국어 UI 문구 수정**
   - 수정 전: "자동 알림을 켜면 모든 변경 사항이 이메일로 발송됩니다."
   - 수정 후(안): "자동 알림을 켜면 모든 변경 사항에 대해 인앱 알림이 전송됩니다."

2. **다국어(EN 등) 동일 문구 여부 확인 및 함께 수정**
   - 영문 UI에서 동일한 오류 문구 존재 여부 점검 필요

3. **릴리즈 노트/변경 로그 반영 여부 검토**
   - 문구 수정이 사용자 경험에 영향을 주는 경우 변경 로그에 기재 권장

## 우선순위 / 참고

- **체감 우선순위**: Medium — 기능 오류는 아니나 사용자 혼동을 직접적으로 유발하는 문구 문제
- **판단 근거**: 문서/카피 수정 성격으로, 코드 변경 최소화 예상. 빠른 반영 가능

  - 이메일 알림 기능이 향후 추가될 예정인지 여부 (로드맵 확인)
